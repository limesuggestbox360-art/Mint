# mint-lang Read-me

## Why to use

Mint programming language or Mtpl provides->

1. Takes less time to master
2. It is easy to use commands
3. Commands which take 10+ lines on python, are 2-3 lines on Mint
4. It is easier to create a module!

## Features

Coloured text
Suggestions
WebBrowserSupport
MediaSupport

## Usage

Open any file with .mint at the end
Type "@" for variables
Type "#" for comments
